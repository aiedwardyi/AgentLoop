# Quality Guidelines

- `2+2=` displays `4`; `9-3=` displays `6`; `6*7=` displays `42`; `8/2=` displays `4`.
- `%` divides the current display value by 100: enter `50`, press `%`, and the display shows `0.5`.
- `C` clears the display to `0`.
- Keyboard works: digits `0-9` and `.` input digits; `+`, `-`, `*`, and `/` select operators; Enter and `=` compute; Escape clears; Backspace deletes the last character.
- Every item above also works via mouse where applicable.
- These items are the complete acceptance criteria.
